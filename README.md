# 🎮 JS Game Hub

A collection of interactive JavaScript mini projects built using **HTML**, **CSS**, and **Vanilla JavaScript**. This project showcases fundamental JavaScript concepts through fun and practical applications.

## 🌐 Live Demo

👉 https://vaghelaindrajeetsinh2006.github.io/js-gamehub/

## 📸 Preview

![JS Game Hub](assets/project-preview.png)

## ✨ Features

- 🎨 Color Scheme Switcher
- 🧮 BMI Calculator
- 🔢 Guess The Number Game
- 🌈 Unlimited Color Generator
- 🎲 Random Color Generator
- ✂️ Stone Paper Scissors Game
- 🔊 Sound Effects
- 📱 Responsive Design
- ⚡ Fast Loading

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/vaghelaindrajeetsinh2006/js-gamehub.git
```

Open the project

```bash
cd js-gamehub
```

Run

Open `index.html` in your browser.
